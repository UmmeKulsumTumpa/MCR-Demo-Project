# Modern Code Review (MCR) Demo Project

## Overview

This repository is a demo project created to practice and demonstrate the Modern Code Review (MCR) process. It showcases the complete review lifecycle, including creating code changes, submitting pull requests, performing peer reviews, addressing feedback, and merging approved changes.

## Purpose

The purpose of this project is to:

* Apply Modern Code Review concepts discussed in class
* Practice collaborative development using version control
* Simulate real-world code review workflows in a team setting

## Project Structure

The project contains a simple, reviewable codebase designed specifically for learning and demonstration purposes. Changes are intentionally kept small to encourage meaningful code reviews and discussions.

## Modern Code Review Process Followed

1. Initial project setup on the main branch
2. Feature development on separate branches
3. Submission of Pull Requests for review
4. Peer code reviews and discussion
5. Code updates based on feedback
6. Final approval and merge

## Tools & Platform

* Version Control: Git
* Code Review Platform: GitHub (Pull Requests)
* Collaboration: Team-based peer reviews
